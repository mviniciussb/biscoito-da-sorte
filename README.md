<h1>BiscoitoSorte</h1>
<br>
<p>Aplicativo biscoito da sorte, onde o usuario no biscoito e o mesmo se abre e mostra uma mensagem aleatória de sorte. Aplicação toda animada criada com HTML, CSS e JavaScript.</p>
